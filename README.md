# CARD_CRM

## First start

* python3 -m venv venv 
* . ./venv/bin/activate
* pip3 install -t requirements.txt
* python card_crm/manage.py makemigrations
* python card_crm/manage.py migrate
* python card_crm/manage.py createsuperuser 
* python card_crm/manage.py runserver
